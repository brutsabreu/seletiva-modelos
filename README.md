## ✨ Sobre o Projeto

Esta é uma **landing page one-page** desenvolvida para captação de candidatas em uma seletiva de moda. O projeto faz parte do meu plano de estudos fullstack e evolui a cada fase com novas tecnologias.

<br>

## 🎯 O Que Ela Faz

| Seção | Descrição |
|:------|:----------|
| 🏠 **Hero** | Chamada principal com imagem e botão de inscrição |
| 📖 **Sobre** | Explicação da seletiva, dados da edição anterior |
| ✅ **Requisitos** | Critérios para participar |
| 💫 **História** | Caso de sucesso com citação e galeria |
| 📝 **Inscrição** | Formulário completo com validação |
| 📜 **Regulamento** | Termos, política e contato |
